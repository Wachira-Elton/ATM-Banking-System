# ATM Banking System

A simple Python-based ATM (Automated Teller Machine) banking system that allows users to perform basic banking operations.

## Features

- **User Authentication**: Secure login with PIN verification
- **Check Balance**: View current account balance
- **Withdraw Funds**: Withdraw money from account
- **Deposit Funds**: Deposit money into account
- **Transfer Money**: Transfer funds to another account
- **Transaction History**: View recent transactions
- **Change PIN**: Update account PIN

## Installation

```bash
python atm_system.py
```

## Usage

1. Run the program
2. Select "Login" to access your account
3. Enter your account number and PIN
4. Choose from the available options
5. Follow the prompts to complete transactions

## Files

- `atm_system.py` - Main ATM application
- `bank_account.py` - Bank account class definition
- `sample_accounts.py` - Sample user accounts for testing
